# HME管理平台

HME管理平台是一个基于Web的内容管理系统，专门设计用于管理公告、通知、活动介绍、积分制内容和戳戳我等多种类型的信息。平台采用纯前端技术实现，数据存储在浏览器本地，无需后端服务器支持。

## 功能特点

### 核心管理功能
- **公告管理**：发布和编辑企业/组织公告
- **通知管理**：发布重要通知信息
- **活动介绍**：管理各类活动介绍内容
- **积分制管理**：分为任务、计划和游戏三个子类别
- **戳戳我内容**：管理互动内容
- **系统设置**：自定义LOGO、背景等平台外观

### 特色功能
- **数据统计**：实时显示各类内容数量统计
- **批量操作**：支持批量清空各类内容
- **数据备份**：支持JSON格式的数据导入/导出
- **密码保护**：管理操作需要密码验证
- **响应式设计**：适配桌面和移动设备
- **图片管理**：支持图片上传、预览和查看
- **附件支持**：积分制内容支持DOCX/PDF附件

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **UI组件**：自定义CSS组件，Font Awesome图标
- **数据存储**：浏览器localStorage
- **图片处理**：原生File API
- **响应式设计**：CSS Grid + Flexbox

## 数据管理

### 数据存储
- 所有数据保存在浏览器的localStorage中
- 数据键名：`hme-data`
- 当前标签页状态：`hme-current-tab`

### 数据备份
1. 点击"导出数据"按钮下载JSON文件
2. 备份文件包含所有内容和设置
3. 使用"导入数据"恢复备份

### 数据安全
- **注意**：清除浏览器缓存会删除所有数据
- 建议定期导出数据备份
- 数据仅存储在本地浏览器，不发送到服务器

## 技术规格

### 支持的文件格式
- **图片**：JPG, JPEG, PNG
- **附件**：DOCX, PDF
- **数据备份**：JSON

### 文件大小限制
- 图片文件：最大5MB
- 附件文件：最大10MB

## 开发说明

### 代码结构
```javascript
// 主要数据结构
appData = {
  announcement: [],      // 公告数组
  notice: [],           // 通知数组
  activity: [],         // 活动数组
  integralTask: [],     // 积分任务数组
  integralPlan: [],     // 积分计划数组
  integralGame: [],     // 积分游戏数组
  poke: [],             // 戳戳我数组
  settings: {           // 设置对象
    logo: "",
    background: "",
    logoDataUrl: "",
    backgroundDataUrl: ""
  }
}
```

### 核心函数
- `loadData()` - 从localStorage加载数据
- `saveData()` - 保存数据到localStorage
- `renderAllData()` - 渲染所有内容
- `applySettings()` - 应用系统设置
- `showToast()` - 显示通知消息

## 自定义开发

### 修改默认密码
在代码中搜索 `DEFAULT_PASSWORD` 变量：
```javascript
const DEFAULT_PASSWORD = "740302"; // 修改此处
```

### 调整样式
主要样式变量定义在 `:root` 选择器中：
```css
:root {
  --primary-color: #4361ee;      /* 主色调 */
  --secondary-color: #3a0ca3;    /* 副色调 */
  --accent-color: #f72585;       /* 强调色 */
  --gradient-primary: linear-gradient(135deg, #4361ee 0%, #3a0ca3 100%);
  /* 其他样式变量... */
}
```

### 添加新的内容类型
1. 在HTML中添加对应的标签页
2. 在JavaScript的 `appData` 对象中添加新数组
3. 更新 `renderAllData()` 和相关函数
